* IOTA Hub
** Dependencies
   - Modern GCC or Clang
   - [[https://github.com/bazelbuild/bazel/releases][bazel]]
   - sqlite3
   - boost
   - python,pyparsing
** How to run
   - `bazel run -c opt //hub:hub -- <parameters>`
  
** Useful things
   - Use `clang-format`
   - Use `protofmt` from `go get -u -v github.com/emicklei/proto-contrib/cmd/protofmt` to format
